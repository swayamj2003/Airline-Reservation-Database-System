# Airline Reservation Database System

## Overview

This project is a MySQL-based Airline Reservation Database System designed to manage airline operations such as flight scheduling, passenger bookings, fare management, crew assignments, itineraries, and payments.

## Database Name

casestudy1

## Tables

* aircraft
* booking
* fare
* flight
* flight_crew
* flight_crew_assignment
* itinerary
* itinerary_flight
* passenger
* payment

## Technologies Used

* MySQL 8.0
* SQL

## Features

* Passenger Management
* Flight Management
* Aircraft Management
* Booking and Reservation Tracking
* Fare Management
* Payment Tracking
* Crew Assignment
* Itinerary Management
